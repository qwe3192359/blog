#### vertical-align
```
永远垂直居中对齐的图标，图片与文字默认对齐在父元素基线（X字母的下边缘baseline）与图片的下边缘，百分比计算也是基于基线
p > img {
 width: 16px; height: 16px;
 vertical-align: 25%;
 vertical-align: .6ex;
 position: relative;
 top: 8px;
} 

```
