#### 函数声明、变量声明提升
* 函数声明、变量声明会提升,赋值不会
* 重复的变量声明、函数声明后面的会被忽略
* 函数声明优先，函数声明提升在变量声明之前，也就是与函数声明同名的变量声明会被忽略
* 函数表达式不会被提升，类似赋值，同名的函数后面的会覆盖前面的
* 变量和赋值语句一起书写，在js引擎解析时，会将其拆成声明和赋值2部分，声明置顶，赋值保留在原来位置
```
console.log(typeof foo);    //function
var foo;
function foo() {
   console.log( 3 );
}
//函数声明优先，无论函数声明在前还是在后，变量类型都会是函数
```

```
foo(); // 3
//先函数声明，后执行函数，所以是3
function foo() {
    console.log( 1 );
}
function foo() {
    console.log( 3 );
}
var foo = function() {
    console.log( 2 );
}
foo(); // 2
//先函数声明，后执行函数，然后赋值，所以是3
```
