#### 常用命令

- 显示版本列表：
  - nvm list
  - nvm ls
- 使用指定版本 node：
  - nvm use <version>
- 安装指定版本 node：
  - nvm install <version>
- 安装最新版 node：
  - nvm install latest
- 安装最新稳定版 node：
  - nvm install lts
- 卸载指定版本 node：
  - nvm uninstall <version>
- 显示当前的 node 版本：
  - nvm cruuent
- 设置 node、npm 镜像：

```
nvm node_mirror https://npmmirror.com/mirrors/node/
nvm npm_mirror https://npmmirror.com/mirrors/npm/
```
